# AI Assist (aiassist.biz) — Next.js + Tailwind Landing

## Quick Start
1) Install Node.js LTS: https://nodejs.org/en/download
2) Install Git: https://git-scm.com/downloads
3) Install dependencies
```bash
npm install
npm run dev
```
Visit http://localhost:3000

## Deploy to Vercel
- Create account: https://vercel.com/signup
- Push this folder to GitHub, then import into Vercel → Deploy.
- Add your domain aiassist.biz in Vercel → Settings → Domains, and follow DNS prompts.

## Customize
- Edit copy in `app/page.jsx`
- Tailwind in `app/globals.css` & `tailwind.config.js`
- Replace logos/branding in `public/`
